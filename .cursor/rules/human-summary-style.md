# Human-facing summaries (portable rule)

When presenting results to **humans** (without changing canonical structured outputs), use:

- a **default short summary** (1–2 sentences);
- an **explicit “interpretation mode”** only if the user asks for deeper comparison/explanation.

**Do not** invent definitions, formulas, or domain meanings: defer to `docs/glossary.md` (and any other normative docs explicitly referenced by the project).
